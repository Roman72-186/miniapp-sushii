Суши-Хаус 39 — Просмотр заказов

Инструкция для администратора

Эта инструкция описывает, как Юлии (администратору) просматривать заказы подписчиков — какие подарочные роллы и сеты были получены, на какой адрес и кем.
1
Открыть страницу администратора

В браузере на телефоне или компьютере перейти по адресу:

https://sushi-house-39.ru/admin

Откроется страница входа. Ввести пароль администратора и нажать «Войти».

📷
Скриншот: страница входа /admin
2
Перейти на вкладку «Заказы»

После входа в верхней части экрана будут видны вкладки. Нажать:

◷ Заказы

Загрузится список всех полученных подарков по подписке.

📷
Скриншот: главная страница админки, вкладки сверху
3
Читать список заказов

Каждая карточка в списке содержит следующую информацию:

Поле Описание
Название Конкретный ролл или сет, который получил клиент
Дата Дата получения подарка
Имя Имя клиента, указанное при оформлении
Телефон Контактный номер клиента
Тариф Подписка клиента — 490 ₽ (роллы) или 1190 ₽ (сеты)
Адрес самовывоза Точка, где клиент забирает заказ
📷
Скриншот: список заказов во вкладке «Заказы»
4
Найти конкретного клиента

Над списком есть поле «Поиск». Можно набрать:

  • Имя клиента
  • Номер телефона
  • Адрес самовывоза

Список сразу отфильтруется по введённому тексту.

Совет: Если нужно найти все заказы по конкретной точке (например, «Гагарина»), достаточно ввести часть адреса в поле поиска.
📷
Скриншот: поле поиска с примером фильтрации
