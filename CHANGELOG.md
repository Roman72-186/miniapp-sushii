# Changelog

Все значимые изменения проекта фиксируются в этом файле.

## [1.0.0] — 2026-04-17

Первая официальная версия. Зафиксировано состояние продакшн-приложения.

### Добавлено
- **Игра «Пятибуквенное слово»** — Wordle на русском для подписчиков (`/game`):
  - Словарь 4397 слов из npm-пакета `russian-words` (фильтр строго 5 кириллических букв)
  - Еженедельный авто-синк словаря (воскресенье 03:00 UTC)
  - Плавающий виджет 🎮 с перетаскиванием и сохранением позиции
  - Flip-анимация ячеек, клавиатура 5 рядов, сохранение прогресса в localStorage
  - Победа = +3 SHC, максимум 3 победы в день
- **Версионирование** — `AppFooter` с номером версии на всех страницах
- **Промокод и порог в корзине** — промокод `102030` (от 2000 ₽) и автоподарок (от 2500 ₽)
- **Скидочный магазин** (`/discount-shop`) — neumorphic редизайн, `shop-v2.css`
- **Форма доставки** — автодополнение адреса (Yandex Geocoder), обязательное поле «Дом», определение ближайшего филиала по координатам
- **Редактирование профиля** — модалка `EditProfileModal` (пользователь и администратор)
- **Поля имени** — `first_name`, `last_name`, `middle_name` в SQLite и PostgreSQL
- **Допродажи в корзине** — `UpsellBlock`, управление из админки (вкладка ◈ Допродажи)
- **Реферальная вкладка в админке** — топ рефереров и история начислений SHC
- **Смена тарифа с datepicker** в карточке подписчика
- **HTTP-кэш изображений** 7 дней в nginx

### Изменено
- Геокодер возвращён на Yandex Geocoder API (был Nominatim)
- Seeding словаря игры — безусловный INSERT OR IGNORE при каждом старте
- Footer вынесен в общий компонент `AppFooter`

### Архитектура
- Standalone веб-приложение (JWT-авторизация по телефону, не Telegram Mini App)
- React 19 (CRA) + Express + SQLite (dev) / PostgreSQL Supabase (prod)
- Docker Compose: `app` + `nginx` + `certbot`
- Продакшн: `https://sushi-house-39.ru`

## Следующие версии

При выпуске новой версии:
1. Обновить `src/version.js` — строка `APP_VERSION`
2. Обновить `package.json` — поле `version`
3. Добавить запись в этот файл
4. Создать git-тег: `git tag v1.1.0 && git push origin v1.1.0`
