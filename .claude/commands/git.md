Автоматический билд, коммит и пуш на GitHub:

1. Собери проект: `npx react-scripts build`
2. Добавь все изменённые и удалённые файлы (кроме .env, credentials, node_modules) в git staging
3. Создай коммит с кратким описанием изменений на русском языке
4. Выполни `git push origin main`
5. Сообщи результат: успех или ошибку
