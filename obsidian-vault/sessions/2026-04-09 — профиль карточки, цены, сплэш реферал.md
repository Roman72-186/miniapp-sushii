# Сессия 2026-04-09 — профиль карточки, новые цены, сплэш по реферальной ссылке

## Что сделано

### 1. Neumorphic редизайн профиля
- `.shop-profile__block` → `.shop-profile__card` (отдельные карточки с gap 10px)
- Добавлен `.shop-profile__card-title` (мини-лейбл teal)
- `.shop-profile__section` теперь внутрикарточный разделитель (border-top subtle)
- Кнопки получили glow-тени (VIP — оранжевый, invite — teal, link-btn — neumorphic)
- Плашка тарифа `.profile-tariff-badge` рядом с именем (490 ₽ / 1190 ₽ / АМБА)

### 2. Реструктуризация ProfilePage — 6 отдельных карточек
Новый порядок:
1. Профиль (имя + badge, телефон)
2. Подписка (статус, дата, подарок, кнопка изменить)
3. Автопродление (♻️ + 💳)
4. История заказов + ⏰ часы работы
5. Реферальная программа (амба или SHC)
6. Кнопки (VIP, поддержка, администратор, настройки)

### 3. Строка тарифа → подарок
- Убрана строка «💎 Тариф» (тариф виден через badge)
- Вместо неё: «🎁 Подарок: ролл/сет доступен сейчас!» или «через N дн.»
- Подарочные окна загружаются на mount (не только при открытии модалки)

### 4. Текст реферальной программы
«Приглашай друзей — получай 20% от их подписки в SHC баллах. 100% заказа оплачивается баллами: накопил 3000 баллов — получил роллов на 3000 ₽»

### 5. Сплэш по реферальной ссылке
- Порядок рендера в LandingPage: loading → splash → login (было: loading → login → splash)
- Теперь все без активной подписки видят промо-фото первым, включая веб-пользователей по `?invited_by=`

### 7. Автопересчёт цен в админке
- `updatePrice` в AdminPage.js: при смене «1 мес» автоматически пересчитываются «3 мес» и «5 мес»
- Коэффициенты: r3 = 750/290 ≈ 2.586 (−13.8%), r5 = 1200/290 ≈ 4.138 (−17.2%)
- Округление до ближайших 10 ₽
- Метки «3 мес ↺» и «5 мес ↺» teal-цветом + подпись «пересчитываются автоматически»
