# 2026-04-17 (7) — Словарь угадывания 6074 слов

## Что сделано

Расширен словарь валидации угадываемых слов в игре «5 букв» с 480 до **6074 слов**.

### Проблема
- Пользователь хотел библиотеку слов минимум 5000 штук
- Один пакет `russian-words` — только 4397 пятибуквенных слов (меньше 5000)
- Только curated-список (480 слов) — слишком жёсткое ограничение

### Решение
Объединение двух пакетов в `Set`:
- `russian-words`: 4397 слов
- `wordlist-russian`: 4731 слов  
- **Union: 6074 уникальных слов** ✅

Оба пакета — леммы (базовые формы = именительный падеж). Соответствует требованию пользователя «только существительные в именительном падеже».

### Архитектура (два словаря)
| Назначение | Источник | Объём |
|------------|----------|-------|
| Загадываемые слова | `game-words.json` (curated) | 480 слов |
| Валидация угадываемых | `russian-words` + `wordlist-russian` | 6074 слова |

`VALID_WORDS` — module-level `Set`, создаётся один раз при старте процесса.

## Файлы изменены

| Файл | Изменение |

| `package.json` | Добавлен `wordlist-russian` в dependencies |
