# Сессия 2026-04-12 — Вкладка «Рефералы» в админке

## Что сделано

Добавлена отдельная вкладка **«⇄ Рефералы»** в группу «Люди» администраторской панели.

### Новый функционал

**Топ рефереров** — список пользователей, у которых есть хотя бы один реферал, отсортированный по количеству рефералов:
- Имя, тариф, телефон, Telegram ID
- Количество рефералов (👥)
- Заработано SHC из referral_bonuses (💎)
- Текущий баланс SHC

**История начислений** — последние 100 записей из `referral_bonuses`:
- Тип: 🤝 join-бонус (friends_count ≥ 1) или 📈 подписочный 20% (friends_count = 0)
- Сумма, описание (achievement)
- Кто получил → от кого
- Дата

**Поиск** — работает одновременно по обоим блокам (имя, телефон, telegram_id).

**Кнопка «Обновить»** — перезагружает данные с сервера.

## SQL-запросы

**Топ рефереров:**
```sql
SELECT u.telegram_id, u.name, u.phone, u.tariff, u.balance_shc,
       (SELECT COUNT(*) FROM users r WHERE r.invited_by = u.telegram_id) AS referrals_count,
       COALESCE((SELECT SUM(rb.total_amount) FROM referral_bonuses rb WHERE rb.user_id = u.telegram_id), 0) AS shc_earned
FROM users u
WHERE (SELECT COUNT(*) FROM users r WHERE r.invited_by = u.telegram_id) > 0
ORDER BY referrals_count DESC
LIMIT 30
```

**История начислений:**
```sql
SELECT rb.*, u.name AS user_name, ref.name AS referral_name
FROM referral_bonuses rb
LEFT JOIN users u ON u.telegram_id = rb.user_id
LEFT JOIN users ref ON ref.telegram_id = rb.referral_id
ORDER BY rb.created_at DESC
LIMIT 100
```
