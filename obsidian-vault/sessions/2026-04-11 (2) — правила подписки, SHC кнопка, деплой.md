## Ловушки

- При `docker compose build --no-cache app` без предварительного `git pull` — берутся старые исходники с VPS. Хеш JS совпадал со старым → изменения не попали в прод.
- Минификатор React экранирует русские строки в Unicode (`\u0421...`), поэтому `grep` по кириллице не работает на минифицированных файлах. Проверять по ASCII-контенту (числа, латиница).
