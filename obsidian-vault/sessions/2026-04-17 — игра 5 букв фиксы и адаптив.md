# 2026-04-17 — Игра «5 букв»: фиксы и адаптив

## Что сделано

### Исправлена внутренняя ошибка сервера при нажатии Enter
- **Причина**: `getGameDailyWord`, `getGameStats`, `recordGameWin` в `db-pg.js` — async-функции, но вызывались без `await` в `game-guess.js` и `game-stats.js`
- `dailyWordRow` был Promise, не объектом → `dailyWordRow.word === undefined` → `TypeError: Cannot read properties of undefined (reading 'toLowerCase')`
- **Фикс**: добавлен `await` в трёх местах
- Коммит: `60e67e4`

### Виджет 🎮 — фикс рандомного положения
- **Причина**: `transform` в CSS-анимации `@keyframes wrd-widget-in` временно создаёт новый containing block → `position: fixed` сбивается на мобиле
- **Фикс 1**: `ReactDOM.createPortal(content, document.body)` — рендер в body, вне любых страничных контейнеров
- **Фикс 2**: убран `transform` из анимации, оставлен только `opacity`
- Коммиты: `1e8161a`, `c08eeb8`

### Виджет — перетаскивание
- Зажал и потянул → перемещение по экрану
- Короткое нажатие (<6px смещения) → переход на `/game`
- Позиция сохраняется в `localStorage['wordle_widget_pos']`
- Позиция зажата в границах экрана (`clampPos`)
- Коммит: `8e47b00`

### Адаптив игрового поля
- `.wrd-cell`: `aspect-ratio: 1` + `width: 100%` вместо фиксированных `52px`
- `.wrd-grid`: `max-width: 320px`
- Шрифт: `clamp(16px, 5vw, 22px)`
- Клавиатура: `flex: 1` + `min-width: 0`, `max-width: min(420px, 100vw-32px)`
- `overflow-x: hidden` на `.wrd-page`
- Коммит: `1e8161a`

### Клавиатура — 5 рядов
- Было: 3 ряда букв + 1 ряд Enter/⌫ = 4 ряда
- Стало: й-х / ф-э / я-ь / б-ю (wide) / Enter-⌫ (action) = 5 рядов
- Ряд «б ю» — `flex: 2` (широкие клавиши)
- Ряд «Enter ⌫» — `flex: 1`, `max-width: none`, `height: 50px`
- Коммит: `431c60b`

### Сохранение прогресса игры
- После каждого засчитанного guess → `localStorage['wordle_progress_{gameDay}']`
- При перезагрузке прогресс восстанавливается
- `resetGame()` очищает запись
- При смене игрового дня → старые ключи удаляются (`cleanStaleProgress`)
- `gameDay` передаётся из `gameStats?.gameDay` (ответ `/api/game-stats`)
- `STORAGE_VERSION = 1` для безопасной инвалидации схемы
- Коммит: `431c60b`
