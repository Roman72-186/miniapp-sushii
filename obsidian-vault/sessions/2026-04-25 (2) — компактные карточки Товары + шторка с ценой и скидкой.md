### Задача 2: Вкладка «Товары» — карточки + шторка

**Frontend (`src/AdminPage.js`):**

Новые состояния:
```js
const [selectedProduct, setSelectedProduct] = useState(null); // {catalogId, index, item}
const [editingProduct, setEditingProduct] = useState({ price: '', discount: '' });
```

useEffect синхронизации selectedProduct при обновлении catalogs (аналог selectedUser/subscribers):
```js
useEffect(() => {
  if (!selectedProduct) return;
  const cat = catalogs.find(c => c.id === selectedProduct.catalogId);
  if (!cat) return;
  const updated = cat.items[selectedProduct.index];
  if (updated) setSelectedProduct(prev => ({ ...prev, item: updated }));
}, [catalogs]);
```

CSS-классы (injected `<style>`):
- `.adm-prod-grid` — 2/3/4 колонки по брейкпоинтам (600/900px)
- `.adm-prod-card` — компактная карточка, overflow hidden
- `.adm-prod-dot.on/.off` — индикатор статуса
- `.adm-prod-name` — название с ellipsis
- `.adm-prod-price` / `.adm-prod-discount` — цена и скидка
- `.adm-prod-tags` — теги «допрод», «🎁 промо», «🎁 авто»

Карточка показывает: статус вкл/выкл, название, цену, скидку (если задана), теги ролей.

Шторка товара (bottom-sheet, z-index 198/199):
- **⚡ Статус** — одна кнопка «Включить» / «Выключить» (зелёная/красная)
- **💰 Цена** — input + кнопка «Сохранить» → showToast
- **🏷 Скидка %** — input 0–99 + кнопка «Сохранить» + кнопка «× Сбросить» если задана
- **⭐ Роль товара** — кнопки Допродажа / Промо-подарок / Авто-подарок (если есть SKU)

`saveItem` теперь возвращает `true` при успехе (для `.then(ok => ok && showToast(...))`)

**Backend (`api/admin-products.js`):**
- GET: добавлено поле `discount` в ответ (из JSON-каталога)
- PUT: принимает `discount` — сохраняет число или удаляет поле если `null`/пустая строка
- Ответ PUT тоже возвращает `discount`

Скидка хранится в JSON-каталоге в `data/products/` (persistent Docker volume).
