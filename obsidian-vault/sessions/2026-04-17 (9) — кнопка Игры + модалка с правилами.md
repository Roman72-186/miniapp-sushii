# 2026-04-17 (9) — кнопка Игры + модалка с правилами

## Задача
На `/discount-shop` добавить пару кнопок над табами-категориями: «🥄 Добавки» (2/3 ширины, скроллит к секции соусов) и «🎮 Игры» (1/3 ширины). Клик по «Игры» открывает модалку со списком игр. Клик по игре показывает описание: правила, условия, награда, кнопка «Играть».

## Решения
- Размещение: новый блок `.shop-quick-row` **над** `<nav.shop-tabs>` (не внутри таб-бара, не в блоке подарков).
- Список игр: только Wordle сейчас, архитектура — массив в `src/config/games.js`, легко расширяется.
- Описания игр: хардкод в `src/config/games.js` (rules, conditions, reward).
- Для не-подписчиков (`profile?.статусСписания !== 'активно'`) кнопка «Играть» disabled с подсказкой.

## Файлы
**Новые:**
- `src/config/games.js` — массив `GAMES` с метаданными и правилами. Одна запись — Wordle (`path: '/game'`).
- `src/components/GamesModal.js` — модалка с двумя состояниями: список игр / описание одной игры. `useEffect` сбрасывает `selectedId` при закрытии.
- `src/games-modal.css` — оверлей, панель (slide-up), сетка карточек, кнопка «Играть», disabled state.

**Изменены:**
- `src/DiscountShopPage.js`:
  - импорт `GamesModal`,
  - `const [gamesOpen, setGamesOpen] = useState(false)`,
  - блок `.shop-quick-row` над `.shop-tabs` (с `!searchQuery` фильтром),
  - рендер `<GamesModal isOpen={gamesOpen} isSubscriber={profile?.статусСписания === 'активно'} />` рядом с `ProductModal`.
- `src/shop-v2.css` — стили `.shop-quick-row` (`grid-template-columns: 2fr 1fr`), `.shop-quick-btn`, `.shop-quick-btn--sauces` (оранжевый акцент), `.shop-quick-btn--games` (зелёный neon).

## Открытые вопросы
- Когда появятся новые игры — просто дописать объекты в `src/config/games.js` (id, path, rules, conditions, reward). Новых роутов не нужно, пока игра сама по себе маршрут-страница.
- Если потребуется админ-редактирование описаний — перейти на `public/games.json` + endpoint (сейчас хардкод).
