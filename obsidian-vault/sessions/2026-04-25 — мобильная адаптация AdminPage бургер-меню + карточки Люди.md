## Что сделано

### Задача
Адаптировать AdminPage под мобильные устройства:
- Все вкладки → всплывающее бургер-меню (bottom-sheet)
- Вкладка «Люди» → сетка карточек пользователей

**Карточки подписчиков (вкладка «Люди»):**
- CSS-класс `adm-user-grid` с медиа-запросами через injected `<style>`
- 1 колонка на узких (< 480px), горизонтальные карточки со стрелкой `›`
- 2 колонки (480–767px), 3 колонки (768–1099px), 4 колонки (≥1100px)
- При ≥480px: вертикальный layout, стрелка скрывается

**Детальная карточка пользователя (bottom-sheet):**
- z-index 298/299 (выше бургер-меню)
- 7 секций: информация, тарифы, подарки (роллы/сеты), SHC, теги, датапикер, заметки, сброс
- Кнопки подарков: `giftBtnAdd` (зелёный, +1) и `giftBtnRemove` (красный, −1) — 2×4 сетка
- useEffect: когда subscribers обновляется → обновляет selectedUser если открыт

**useEffect синхронизации selectedUser:**
```js
useEffect(() => {
  if (!selectedUser) return;
  const updated = subscribers.find(s => s.telegram_id === selectedUser.telegram_id);
  if (updated) setSelectedUser(updated);
}, [subscribers]);
```

### Медиа-запросы (injected `<style>`)

```css
@media (min-width: 480px) { 1fr 1fr }                      /* телефон горизонт */
@media (min-width: 768px) { 1fr 1fr 1fr }                  /* планшет */
@media (min-width: 1100px) { repeat(4, 1fr) }              /* десктоп */
```
