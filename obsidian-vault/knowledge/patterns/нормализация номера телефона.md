# Нормализация номера телефона

## Правило

Везде в системе телефон хранится и передаётся в формате **`7XXXXXXXXXX`** — без плюса, 11 цифр.

Исключение — чек YooKassa (требование 54-ФЗ): `+7XXXXXXXXXX` с плюсом.

## Алгоритм нормализации

```js
function normalizePhone(raw) {
  const nums = String(raw || '').replace(/\D/g, '');  // только цифры
  if (nums.length === 11 && nums.startsWith('8')) return '7' + nums.slice(1);
  if (nums.length === 11 && nums.startsWith('7')) return nums;
  if (nums.length === 10) return '7' + nums;
  return nums;  // нестандартный — возвращаем как есть
}
```

Примеры:
- `+7 (900) 123-45-67` → `79001234567`
- `8-900-123-45-67` → `79001234567`
- `9001234567` → `79001234567`
