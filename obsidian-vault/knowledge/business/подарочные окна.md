# Подарочные окна

## Что это

Механизм, позволяющий подписчикам получить бесплатный ролл или сет в течение определённого периода.

| Тариф | Подарок | Периодичность |
|-------|---------|--------------|
| 490 | 1 ролл | каждые 15 дней |
| 1190 | 1 сет | каждые 30 дней |

## Хранение

Blob-store: `data/gifts/{telegram_id}.json`
Формат: массив «окон» с датами и статусом.

```json
{
  "windows": [
    {
      "num": 1,
      "start": "01.04.2026",
      "end": "15.04.2026",
      "status": "available"
    },
    {
      "num": 2,
      "start": "16.04.2026",
      ...
      "status": "claimed",
      "claimedAt": "03.04.2026"
    }
  ],
  "updatedAt": "2026-04-03T10:00:00.000Z"
}
```

## Генерация окон

`buildWindows(start, end, windowDays)` в `api/_lib/gift-windows.js`
- `start` — дата начала подписки
- `end` — дата окончания
- `windowDays` — 15 или 30

## API

| Эндпоинт | Описание |
|---------|---------|
| `GET /api/get-gift-windows` | Текущий статус окон пользователя |
| `POST /api/claim-gift` | Отметить подарок полученным |
| `POST /api/admin/grant-gift` | Выдать подарок из админки |
| `POST /api/admin/claim-gift` | Отметить полученным из админки |

## Статусы окна

| Статус | Значение |
|--------|---------|
| `available` | Можно получить прямо сейчас |
| `claimed` | Уже получен |
| (нет текущего окна) | Следующее ещё не наступило |

## Admin grants

Администратор может выдать подарок в обход тарифа:
```json
{ "grantedBy": "admin", "grantType": "roll" | "set" }
```
`adminGrants` приходят в ответе `get-gift-windows` и отображаются отдельно в `DiscountShopPage`.

## Страницы

- `/gift-rolls` → `GiftRollsPage` — страница для получения ролла (ссылка из бота)
- `/gift-sets` → `GiftSetsPage` — страница для получения сета
