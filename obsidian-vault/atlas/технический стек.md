# Технический стек

## Frontend

| Технология | Версия | Роль |
|-----------|--------|------|
| React | 19 | UI-фреймворк |
| CRA (Create React App) | — | Сборка, dev-сервер |
| Plain CSS | — | Стили (`shop.css`, `App.css`) |
| Telegram WebApp SDK | — | Идентификация, тема, кнопки |

Нет: react-router, Redux, TypeScript, CSS-модулей.

## Backend

| Технология | Версия | Роль |
|-----------|--------|------|
| Node.js | 20 (Alpine) | Рантайм |
| Express | — | HTTP-сервер + API |
| better-sqlite3 | — | Синхронный SQLite-клиент |
| dotenv | — | Переменные окружения |

## Хранилища

| Хранилище | Путь | Назначение |
|-----------|------|-----------|
| SQLite | `data/sushii.db` | Основная БД (пользователи, платежи, транзакции) |
| Файловый кэш | `data/users/{id}.json` | Кэш пользователя (TTL 5 мин) |
| Blob-store | `data/gifts/{id}.json` | Подарочные окна |
| Продуктовые оверрайды | `data/products/` | Переопределения цен от админки |
| Баннеры | `data/banners/` | Загруженные через админку картинки |

## Внешние сервисы

| Сервис | Переменные окружения | Назначение |
|--------|---------------------|-----------|
| Frontpad | `FRONTPAD_SECRET` | CRM, заказы, каталог товаров |
| YooKassa | `YOOKASSA_SHOP_ID`, `YOOKASSA_SECRET_KEY` | Приём платежей |
| Telegram Bot API | `TELEGRAM_BOT_TOKEN` | Уведомления, кнопки |
| Supabase | `SUPABASE_URL`, `SUPABASE_SERVICE_KEY`, `SUPABASE_ANON_KEY` | Пароли веб-входа (`web_credentials`) |

→ [[Supabase веб-авторизация]] — Supabase используется ТОЛЬКО для паролей, не для основных данных.

## Команды

```bash
npm start              # React dev-сервер (:3000)
npm run build          # Продакшн-сборка
node server.js         # Только бэкенд (:3001)
npm run fetch:products # Получить товары из Frontpad
npm run backup:db      # Бэкап SQLite
npm test               # Базовый тест рендера (App.test.js)
```
