## Быстрый старт

```bash
npm start          # фронт на :3000
node server.js     # бэк на :3001
npm run build      # продакшн-сборка
```

**Деплой на VPS:**
```bash
ssh root@64.188.63.249
cd miniapp-sushii && git pull && docker compose up -d --build
```

## Ключевые файлы

| Файл | Роль |
|------|------|
| `src/App.js` | Роутинг (без react-router, через `window.location.pathname`) |
| `src/UserContext.js` | Глобальный стейт пользователя |
| `src/DiscountShopPage.js` | Главная страница: скидки + подарки |
| `server.js` | Express-сервер + cron подписок |
| `api/_lib/db.js` | SQLite: все операции с БД |
| `api/_lib/subscription-state.js` | Вычисление статуса подписки |
| `src/config/imageMap.js` | Маппинг названий товаров → картинки |
| `data/catalog-master.csv` | Единый справочник товаров (создан 2026-04-07) |
