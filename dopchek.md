Проанализируй существующий React-проект и внедри блок допродаж в корзину.

Нужно:
- добавить внизу корзины отдельный блок рекомендованных товаров;
- вывести 3–4 позиции;
- каждая позиция должна иметь фото, название, цену и кнопку "Добавить";
- товар должен добавляться в корзину одной кнопкой;
- сумма заказа должна обновляться сразу;
- не использовать переходы на страницу товара;
- не ломать существующую бизнес-логику корзины;
- использовать текущие функции addToCart / updateCart, если они уже есть;
- вынести новый UI в отдельный компонент;
- сделать решение адаптивным для мобильного интерфейса;
- использовать существующие стили проекта или сделать минимально совместимые.

Источник данных для допродаж:
- сначала используй простой конфиг с массивом ID товаров;
- затем через существующий каталог подтяни данные по этим ID;
- исключи недоступные товары;
- исключи позиции, которых нет в наличии;
- максимум 4 товара.

Важно:
- не переписывать корзину целиком;
- не создавать сложную recommendation-систему;
- сделать быстрое и устойчивое решение, которое можно масштабировать;
- после выполнения покажи:
  1. какие файлы изменены,
  2. какой компонент добавлен,
  3. как работает логика,
  4. какие следующие улучшения можно сделать.
